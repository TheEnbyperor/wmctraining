Allows to show announcements in your blog. Show/Hide announcement for specific days, automatically with end date. Highly customizable

== Description ==

This a extremely simple but more powerful announcement plugin . Just easy way to create a announcement as easy as 1-2-3.

Supports HTML announcements. Cookie enabled. Automatically closes the announcement if the expiry date reaches. No programming skills needed. Can use custom class, IDs for the announcements.

Just need to enter the announcement, enter the expiry date and click ON. That's all.

= Features =

* Inbuilt timer for switching on/off the announcement.
* Tested with major browsers like IE 7,8,  Firefox 3>, Google chrome, Safari, Opera for javascript and css support. 
* Can create custom css for announcements.
* Can stop showing the annoncement anytime.
* WYSIWYG editor for editing announcements.
* Contains 2 inbuilt classes (see screenshots).
* Cookie enabled in hiding the announcement.
* Can disable or enable cooking or showing close button.
* Expires automatically when the expiry date reaches.
* Announcement can be added to any page any area
* Can add Styles to the inbuilt style list
* Easy admin page (see screenshots).
* 1-2-3 steps to create a announcement.

== Installation ==

1. Extract the zipped file and upload the folder `Announcer` to to `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to the Announcer admin page through Settings -> Announcer.
1. That's all.

= Usage = 

1. Install and activate.
1. Add the PHP function `<?php if(function_exists('announcer')) { announcer();} ?>` in your theme where ever or use the option in the admin menu to automatically place the announcement in the page.
1. Goto to Announcer admin panel.
1. Enter the announcement.
1. Enter the expiry date, div box's class etc.
1. That's all.
